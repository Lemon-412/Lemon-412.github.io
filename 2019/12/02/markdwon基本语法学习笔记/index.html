markdwon基本语法学习笔记

发表于 2019-12-02

Markdown是一种非常好学的工具语言，只要掌握这些基本内容，就算快速入门了。

1. 标题<\center>

在文字前加“#”来区分标题级别，最高支持六级标题

2. 字体<\center>

使用“*”倾斜文字 this is a test text
使用“**”加粗字体 this is a test text
使用“~~”删除文字 this is a test text

3. 引用<\center>

在文字前加“>”来表示引用

据说“>”可以无限追加下去

Markdown is easy to learn!
